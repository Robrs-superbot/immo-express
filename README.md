# Immo-Express 🏠

**Plateforme immobilière intelligente** — Vente & Location

Immo-Express est une plateforme web qui connecte directement vendeurs et acheteurs, particuliers et professionnels. Pas d'intermédiaire, que de l'immobilier pur, avec une intelligence artificielle qui sécurise et simplifie chaque transaction.

---

## ✨ Fonctionnalités innovantes

| # | Fonctionnalité | Description |
|---|---|---|
| 🛡️ | **Score de confiance** | Chaque annonce notée sur 100 : identité, photos, prix, qualité |
| 📊 | **Analyse IA du prix** | Comparaison automatique avec le marché local |
| 📉 | **Historique des prix** | Toutes les baisses de prix affichées en transparence |
| 🤝 | **Smart Match** | Score de compatibilité entre le bien et les critères de l'acheteur |
| 📅 | **Réservation de visites** | Créneaux disponibles directement dans l'annonce |
| 🎨 | **IA Décoration** | Visualisation du bien aménagé dans 4 styles |
| 💬 | **Assistant IA** | Chatbot qui guide et répond à toutes les questions |
| 📝 | **Dépôt d'annonce** | Formulaire complet en 8 sections |
| ❤️ | **Favoris** | Sauvegarde des biens, stocké localement |
| 📋 | **Récemment consultés** | Historique de navigation |
| 🔗 | **Partage** | Lien, email, WhatsApp |
| 💰 | **Calculateur prêt** | Simulation mensualités |

## 🚀 Pour commencer

Ouvrez simplement `index.html` dans votre navigateur.


## 💡 Concept

> *"Une plateforme 100% immobilière, pas un site d'annonces généralistes."*

**Particuliers** : gratuit, dépôt en 5 minutes, contact direct
**Professionnels** : abonnement, API, tableau de bord, visibilité renforcée

L'innovation clé : **le Score de Confiance IA** qui analyse chaque annonce et attribue une note sur 100, permettant aux acheteurs de gagner un temps considérable en filtrant les annonces de qualité.
